# 基于 FreeRTOS 的智能像素时钟

## 项目概述

| 属性 | 内容 |
|------|------|
| **开发时间** | 2025.11 - 2025.12 |
| **主控芯片** | STM32F407VGT6 (Cortex-M4 @168MHz) |
| **实时系统** | FreeRTOS |
| **代码规模** | 约 5,000 行 C 代码 |
| **项目角色** | 独立开发 |

## 技术栈

- **MCU**: STM32F407 (Cortex-M4 @168MHz, 1MB Flash, 192KB SRAM)
- **RTOS**: FreeRTOS (11个任务并发运行)
- **通信**: ESP8266 WiFi 模块、HTTP 协议、JSON 解析
- **显示**: WS2812B LED 矩阵 (DMA+PWM 驱动)
- **传感器**: AHT20 温湿度传感器 (I2C)
- **存储**: AT24C512 EEPROM (I2C)

## 硬件设计

### PCB 设计

- **层数**: 4 层板设计
- **集成模块**:
  - WS2812B LED 矩阵驱动电路
  - ESP8266 WiFi 模块接口
  - AHT20 温湿度传感器接口
  - AT24C512 EEPROM 接口
  - 电源转换电路 (5V→3.3V)
- **设计考虑**: EMC 布局、散热设计、信号完整性

### 3D 外壳设计

- **设计软件**: SolidWorks
- **设计特点**:
  - 散热风道设计
  - 按键开孔布局
  - 装配公差控制 (±0.2mm)
- **加工方式**: FDM 3D 打印

## 软件设计

### FreeRTOS 任务列表

| 任务名称 | 优先级 | 功能描述 |
|----------|--------|----------|
| vTaskTimeSync | High | NTP 时间同步 |
| vTaskWeather | Medium | HTTP 天气获取 |
| vTaskDisplay | High | LED 矩阵刷新 |
| vTaskAnimation | Medium | 动画效果渲染 |
| vTaskSensor | Low | 温湿度采集 |
| vTaskStorage | Low | EEPROM 读写 |
| vTaskNetwork | High | 网络状态管理 |
| vTaskUI | Medium | 用户交互处理 |
| vTaskSystem | Low | 系统监控 |

### 关键技术

1. **DMA+PWM 驱动 WS2812B**
   ```c
   // DMA 传输实现零 CPU 占用的 LED 刷新
   void WS2812B_DMA_Update(uint32_t* color_data, uint16_t num_leds);
   ```
   - 刷新率: 66Hz
   - CPU 占用: 0% (纯 DMA 传输)

2. **ESP8266 AT 命令栈封装**
   - 支持 TCP/UDP 连接
   - HTTP GET/POST 请求
   - 自动重连机制

3. **NTP 毫秒级对时**
   
   - 支持多个 NTP 服务器
   - 自动时区转换
   - 闰秒处理
   
4. **网络故障自修复**
   - 连接断开自动重连
   - 数据包丢失重传
   - 故障降级显示
