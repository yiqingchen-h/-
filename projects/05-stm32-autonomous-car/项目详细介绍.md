# STM32 自动行驶小车

## 项目概述

| 属性 | 内容 |
|------|------|
| **开发时间** | 2024.08 |
| **主控芯片** | STM32F103C8T6 |
| **姿态传感器** | MPU6050 (DMP) |
| **项目角色** | 独立开发 |

## 技术栈

- **MCU**: STM32F103C8T6 (Cortex-M3 @72MHz)
- **姿态解算**: MPU6050 + DMP 数字运动处理器
- **控制算法**: PID 平衡控制
- **电机驱动**: PWM 控制 (10kHz)
- **导航**: 红外巡线传感器
- **显示**: OLED 状态显示

## 核心功能

### 系统架构

```
┌────────────────────────────────────────────┐
│              导航决策层                     │
│         巡线传感器 → 路径规划               │
├────────────────────────────────────────────┤
│              姿态控制层                     │
│    MPU6050 → DMP解算 → PID → PWM输出       │
├────────────────────────────────────────────┤
│              执行层                        │
│         电机驱动 → 轮式运动                │
└────────────────────────────────────────────┘
```

### 关键算法

1. **DMP 姿态解算**
   - 内置数字运动处理器
   - 输出四元数/欧拉角
   - 6轴传感器融合

2. **PID 平衡控制**
   ```c
   float balance_pid(float angle, float gyro);
   ```

3. **巡线算法**
   - 多路红外传感器
   - 偏差计算
   - 方向修正

## 硬件设计

### PCB 设计

- **层数**: 2 层控制底板
- **集成模块**:
  - 电机驱动电路 (双路 H 桥)
  - MPU6050 传感器接口
  - 巡线传感器接口
  - 电源管理电路

### 关键器件

| 器件 | 型号 | 功能 |
|------|------|------|

| 姿态传感器 | MPU6050 | 6轴运动检测 |
