# 基于 STM32H7 的多旋翼飞控系统开发与调试

## 项目概述

| 属性 | 内容 |
|------|------|
| **开发时间** | 2025.09 - 2025.10 |
| **项目类型** | 多旋翼飞控系统搭建与调试 |
| **参考方案** | WFG100A 开源飞控资料 |
| **主控芯片** | STM32H743VIT6 |
| **飞控固件** | APM |
| **项目角色** | 独立复刻 |

## 技术栈

- **MCU**: STM32H743VIT6
- **飞控固件**: APM
- **地面站**: Mission Planner
- **烧录工具**: STM32CubeProgrammer
- **烧录方式**: Boot0 + USB、DFU 下载
- **固件格式**: `.bin`、`.hex`、`.elf`
- **惯性测量**: ICM-42688-P、ICM-42605
- **磁力计**: IST8310
- **气压计**: SPL06-001
- **扩展模块**: GPS、接收机、遥控链路、电机输出接口、TF 卡、OSD、CAN 通信

## 系统架构

```text
Mission Planner 地面站
        │
        ▼
APM 飞控固件参数配置
        │
        ▼
STM32H743VIT6 飞控主控
        │
        ├── IMU 数据采集
        ├── 磁力计与气压计采集
        ├── GPS 与接收机输入解析
        ├── PID 参数整定
        ├── 电机输出控制
        └── 试飞联调验证
```

## 硬件设计

### 硬件组成
- STM32H7 飞控主控板
- 双 IMU 惯性测量单元
- 磁力计与气压计模块
- USB-C 下载与调试接口
- 接收机与遥控输入接口
- GPS 扩展接口
- 电机输出接口
- TF 卡与 OSD 扩展模块
- CAN 通信接口

### 关键器件

| 器件 | 型号 | 功能 |
|------|------|------|
| 主控 MCU | STM32H743VIT6 | 飞控主处理器 |
| IMU 1 | ICM-42688-P | 姿态与角速度检测 |
| IMU 2 | ICM-42605 | 冗余惯性测量 |
| 磁力计 | IST8310 | 航向检测 |
| 气压计 | SPL06-001 | 高度测量 |
| OSD 芯片 | AT7456E | 图传字符叠加 |
| CAN 收发器 | TJA1051TK/3 | CAN 总线通信 |
| 存储扩展 | TF 卡座 | 日志与数据存储 |
| 下载接口 | USB-C | 固件烧录与调试 |

## 软件设计

### 核心功能
1. **飞控固件烧录**
   使用 STM32CubeProgrammer 完成飞控固件下载，并通过 `Boot0 + USB` 方式进入下载模式完成烧录。

2. **传感器系统联调**
   完成 IMU、磁力计、气压计等核心飞控传感器的识别、接线检查和基础联调，确保飞控具备姿态感知与环境测量能力。

3. **接收机与遥控链路配置**
   完成接收机、遥控输入与地面站对应配置，使飞控能够正确响应遥控指令并进入调试与飞行状态。

4. **PID 参数整定**
   结合 Mission Planner 对飞控参数进行整定，围绕姿态稳定性、控制响应和飞行手感完成多轮调参。

5. **电机输出与试飞验证**
   完成电机输出检查、基础联调与整机试飞验证，确认系统从烧录、配置到飞行链路均能正常工作。

### 调试内容
- 固件烧录与版本部署
- Mission Planner 地面站连接与参数配置
- IMU 校准
- 磁力计校准
- 气压计相关联调
- 接收机通道配置
- 遥控输入联调
- GPS 模块联调
- 电机输出测试
- PID 参数整定
- 实际试飞验证

### 关键技术
1. **STM32 飞控部署链路搭建**
   打通基于 STM32H7 主控的飞控固件烧录流程，掌握 Boot0 下载模式与固件部署方法。

2. **多传感器模块联调**
   对 IMU、磁力计、气压计、GPS、接收机等模块进行逐项整定，完成飞控基础感知链路构建。

3. **基于地面站的参数整定**
   借助 Mission Planner 对飞控参数进行配置与调整，重点完成 PID 相关整定与飞行前联调。

4. **整机试飞验证**
   项目最终完成试飞，说明硬件焊接、固件烧录、参数配置和系统联调已经形成完整闭环。
