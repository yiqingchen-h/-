# 版本 v1.1 - 添加舵机云台电磁铁

## 版本概览

在v1.0基础循迹功能的基础上，v1.1版本增加了物体抓取功能，实现了第一次题目的完整功能。

## 主要功能

### 新增功能
1. **PWM舵机控制**：标准舵机控制，90°回正初始化
2. **步进云台控制**：云台精确定位功能
3. **电磁铁控制**：物体抓取功能（PB17引脚控制）
4. **物体操作**：抓取、移动、放置完整流程

### 原有功能增强
- **循迹算法优化**：PID参数调整
- **运动控制改进**：云台与底盘运动协调
- **系统稳定性**：增加错误处理机制

## 硬件配置

### 新增外设
- **PWM舵机**：标准舵机，角度控制
- **步进云台**：云台定位系统
- **电磁铁**：物体抓取执行器
- **控制电路**：电磁铁驱动电路

### 引脚扩展
- 舵机控制：PWM输出引脚
- 电磁铁控制：PB17 GPIO引脚
- 云台控制：步进电机控制引脚

### 系统集成

## 控制算法

### 动作规划
1. **接近目标**：循迹到目标位置
2. **云台定位**：调整云台到抓取位置
3. **电磁铁激活**：抓取物体
4. **运输移动**：携带物体移动
5. **放置物体**：释放物体到目标位置

### 时序控制
- **动作间隔**：防止机械冲突
- **状态验证**：确保前序动作完成
- **错误恢复**：动作失败处理机制

## 硬件连接

### 新增连接
1. **舵机连接**：PWM信号线 + 电源线
2. **电磁铁连接**：驱动板控制线
3. **云台电源**：独立电源供电

### 电源管理
- **电机电源**：12V/2A
- **控制电源**：3.3V/1A
- **舵机电源**：5V/3A（建议单独供电）
