## 核心算法

### 双PID控制器
1. **steering_Pid**：正常循迹控制
   - Kp = 4.7, Kd = -0.8
   - 误差累积防止积分饱和
2. **revolve_Pid**：原地转向控制
   - Kp = 10, Kd = 0
   - 结合IMU数据提高精度

### 路径决策逻辑
- **灰度值0x0A**：左转标志
- **灰度值0x0B**：右转标志
- **灰度值0xE**：结束停止标志
- **特殊路径处理**：减速区、转弯区识别

### 定时器中断服务
- **Trace_Timer_INST**：主控制循环（10ms周期）
  - 灰度传感器读取
  - PID计算和电机控制
  - 云台动作调度
- **TIMER_0_INST**：辅助定时功能

## 完整搬运流程

### 工作流程
1. **系统初始化**：各模块自检和初始化
2. **等待指令**：树莓派发送任务指令
3. **路径导航**：循迹到目标位置
4. **物体识别**：等待视觉识别结果
5. **抓取操作**：云台定位+电磁铁激活
6. **运输移动**：携带物体到目标区域
7. **放置操作**：精确定位并释放物体
8. **返回待命**：回到待机位置

### 状态管理
- **就绪状态**：等待指令
- **执行状态**：任务执行中
- **暂停状态**：临时停止
- **错误状态**：故障处理
- **完成状态**：任务完成
