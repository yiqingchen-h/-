# 版本 v1.0 - 基础循迹完成

## 版本概览

这是智能搬运小车控制系统的第一个完整版本，实现了基础的循迹功能。

## 主要功能

### 核心功能
1. **灰度传感器路径识别**：使用8通道灰度传感器阵列检测路径
2. **PID循迹算法**：基础PID算法实现精确路径跟踪
3. **步进电机控制**：TMC2209驱动芯片控制左右轮运动

### 技术特点
- **基础运动控制**：直线和曲线循迹
- **实时控制**：10ms控制周期
- **模块化设计**：初步的硬件抽象层设计

## 硬件配置

### 主要外设
- **微控制器**：TI MSPM0G3507
- **传感器**：灰度传感器阵列（I2C接口）
- **电机驱动**：TMC2209步进电机驱动芯片
- **电源**：12V电机供电，3.3V控制供电

### 引脚分配
- 灰度传感器：I2C接口（SCL/SDA）
- 步进电机：Step/Direction控制引脚
- 调试接口：UART0串口

## 软件架构

### 文件结构
```
Stepping_Trace_1.0/
├── SYS/           # 系统层
├── HardWare/      # 硬件驱动层
├── Stepping_Trace.c # 主程序
└── 配置文件
```

### 关键算法
- **PID控制器**：比例-微分控制（Kp=4.7, Kd=-0.8）
- **误差计算**：基于灰度传感器读数的路径偏差
- **电机控制**：速度闭环控制
