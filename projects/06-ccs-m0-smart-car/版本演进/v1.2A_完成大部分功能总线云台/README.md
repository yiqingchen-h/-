# 版本 v1.2A - 完成大部分功能（总线云台）

## 版本概览

v1.2A版本在v1.1基础上进行了重要升级，将云台系统升级为总线云台，并增加了IMU惯性测量单元，提高了系统的精度和可靠性。

## 主要功能

### 重大升级
1. **总线云台替换**：将PWM舵机云台升级为总线云台
2. **IMU集成**：增加惯性测量单元，提供姿态信息
3. **控制算法优化**：针对总线云台的重量调整PID参数

### 功能增强
- **云台控制精度提升**：总线通信提供更精确的控制
- **姿态感知能力**：IMU提供实时姿态数据
- **系统可靠性**：总线云台减少机械误差

## 硬件配置

### 关键升级
- **总线云台**：双舵机协同控制，PWM范围1700-2500
- **IMU模块**：惯性测量单元，I2C接口
- **控制电路**：总线云台驱动电路

### 引脚变化
- 云台控制：从PWM引脚改为总线通信引脚
- IMU接口：新增I2C设备（与灰度传感器共享总线）
- 电源调整：总线云台可能需要不同的电压

## 已知问题与解决方案

### 问题1：总线云台响应延迟
- **现象**：云台动作有轻微延迟
- **原因**：总线通信协议开销
- **解决方案**：优化通信时序，预计算动作

### 问题2：IMU数据漂移
- **现象**：长时间运行后角度漂移
- **原因**：传感器累积误差
- **解决方案**：定期零位校准，互补滤波

### 问题3：电源干扰
- **现象**：总线通信受电机干扰
- **原因**：电源噪声
- **解决方案**：增加电源滤波，分离供电
