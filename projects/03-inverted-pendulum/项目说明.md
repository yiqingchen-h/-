# 倒立摆控制系统

## 项目概述

| 属性 | 内容 |
|------|------|
| **开发时间** | 2024.09 - 2024.10 |
| **主控芯片** | STM32F103C8T6 |
| **当前控制思路** | 角度稳定 + 位置回中调试 |
| **代码规模** | 约 2,000 行 C 代码 |
| **项目角色** | 独立开发 |

## 技术栈

- **MCU**: STM32F103C8T6 (Cortex-M3 @72MHz)
- **传感器**: 电位器角度采样 + 增量式编码器
- **执行器**: 直流电机 + PWM 驱动
- **控制算法**: PID 调试
- **显示与调试**: OLED + 串口输出

## 硬件组成

| 模块 | 说明 |
|------|------|
| 主控 | STM32F103C8T6 |
| 电机驱动 | 直流电机 PWM 驱动 |
| 角度采样 | ADC 采集摆杆角度 |
| 位置反馈 | 编码器测速与位移累计 |
| 显示 | OLED 128x64 |
| 调试接口 | UART 串口 |

## 软件结构

### 主循环部分

- 按键启动 / 停止控制
- 记录当前角度零点
- 调整目标位置
- OLED 显示调试参数

### 定时控制部分

- 周期读取角度与编码器数据
- 累计位移量
- 根据目标位置生成目标速度
- 角度环和速度环共同输出控制量
- 倾倒时关闭输出并置错误标志

## 完成内容

- PWM 输出与电机基本驱动
- 编码器读取与速度计算
- ADC 角度采样
- 定时中断控制框架
- 启停逻辑和零点记录
- 超范围保护停机
- OLED 与串口调试输出
