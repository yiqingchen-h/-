# RA6M5 智能机器人控制系统

## 项目概述

| 属性 | 内容 |
|------|------|
| **开发时间** | 2026.03 - 2026.04 |
| **主控芯片** | Renesas RA6M5 (Cortex-M33 @200MHz) |
| **控制框架** | 状态机 + 定时中断调度 |
| **代码规模** | 约 9,000 行 C 代码 |
| **项目角色** | 除了jeston nano部分 独立开发 |

## 技术栈

- **MCU**: Renesas RA6M5 (Cortex-M33 @200MHz, 2MB Flash, 512KB SRAM)
- **传感器**: 激光雷达 (UART)、编码器 (正交解码)
- **执行器**: 6DOF 机械臂 (逆运动学控制)
- **通信**: UART 协议设计、视觉模块协同
- **算法**: 滑动窗口滤波、逆运动学解算、PID 控制
- **软件组织**: 主循环慢任务 + 1ms/10ms 定时任务 + 运行状态机

## 硬件设计

### PCB 设计

- **层数**: 2 层机器人控制底板
- **集成模块**:
  - TB6612 双路电机驱动电路
  - 激光雷达 UART 接口
  - 编码器正交解码接口
  - 电源管理模块 (5V/3.3V 稳压)
  - 机械臂舵机控制接口
- **设计考虑**: EMC 布局、机械安装孔位、散热设计

### 关键器件

| 器件 | 型号 | 功能 |
|------|------|------|
| 主控 | RA6M5 | 系统控制核心 |
| 电机驱动 | TB6612FNG | 直流电机驱动 |
| 激光雷达 | RPLIDAR A1 | 环境感知 |
| 机械臂 | 6DOF 舵机臂 | 执行抓取 |

## 软件设计

### 控制流程

| 模块 | 触发方式 | 功能 |
|------|----------|------|
| 主循环 | 持续轮询 | 菜单刷新、按键处理、慢速任务调度 |
| 1ms 定时中断 | GPT 定时器 | 按键节拍、电压采样计数 |
| 10ms 快任务 | 由 1ms 节拍分频触发 | 运动控制与核心协调 |
| 运行状态机 | 全局状态变量 | 小车启停、导航执行、动力输出切换 |

### 关键算法

1. **激光雷达滑动窗口滤波**

   ```c
   float sliding_window_filter(float new_data, float* buffer, int size);
   ```

2. **6DOF 机械臂逆运动学**

   ```c
   int inverse_kinematics(float x, float y, float z, float* angles);
   ```

3. **分层协调控制**

- 定时中断只保留节拍和快速调度
- 主循环负责界面、输入和慢速逻辑
- 小车运行由状态机统一协调
