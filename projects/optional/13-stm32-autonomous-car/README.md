## 项目概述

| 属性 | 内容 |
|------|------|
| **开发时间** | 2024.08 |
| **主控芯片** | STM32F103C8T6 |
| **姿态传感器** | MPU6050（DMP） |
| **项目角色** | 独立开发 |

## 技术栈

- **MCU**：STM32F103C8T6（Cortex-M3 @72MHz）
- **姿态解算**：MPU6050 + DMP 数字运动处理器
- **控制算法**：PID 平衡控制
- **电机驱动**：PWM 控制（10kHz）
- **导航方式**：红外巡线传感器
- **显示方式**：OLED 状态显示

## 系统架构

```text
导航决策层
    └── 巡线传感器 -> 路径修正
姿态控制层
    └── MPU6050 -> DMP 解算 -> PID -> PWM 输出
执行层
    └── 电机驱动 -> 轮式运动
```

## 核心功能

1. **DMP 姿态解算**  
   利用 MPU6050 内置数字运动处理器输出姿态角与角速度数据，减轻主控计算负担。

2. **PID 平衡控制**  
   基于姿态角和角速度完成动态平衡调节，输出双轮驱动控制量。

3. **巡线控制**  
   通过红外巡线传感器识别路径偏差，并在平衡控制基础上叠加方向修正。

4. **OLED 状态显示**  
   显示姿态、控制状态和基础运行信息，便于联调。
