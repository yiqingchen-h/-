# RTOS 多串口通信系统

## 项目概述

基于 FreeRTOS 的三路高速 UART 通信系统，展示实时系统中的串口管理和数据缓冲技术。

## 技术栈

- **主控**: STM32F103C8T6
- **RTOS**: FreeRTOS
- **通信**: 3x UART (最高921600bps)
- **缓冲**: 环形缓冲区 (Ring Buffer)

## 核心功能

- 三路 UART 并发通信
- 环形缓冲区数据管理
- DMA + 中断传输
- 数据回声测试

## 性能指标

| 参数 | 数值 |
|------|------|
| 最高波特率 | 921600 bps |
| 缓冲区大小 | 512B/路 |
| CPU占用 | <5% |
