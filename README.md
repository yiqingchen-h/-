# 嵌入式开发项目集

## 个人简介
- 方向：嵌入式软件开发（单片机 / RTOS / 控制 / 通信 / 视觉联动）
- 技能：STM32、Renesas RA6M5、TI MSPM0、树莓派、FreeRTOS、C 语言、Python、OpenCV、MediaPipe、PCB 设计、3D 建模

## 技术栈
- 微控制器：STM32F1/F4/H7、Renesas RA6M5、TI MSPM0G3507
- 边缘平台：Raspberry Pi
- RTOS：FreeRTOS
- 通信协议：UART、I2C、SPI、HTTP、JSON
- 视觉处理：OpenCV、MediaPipe、OpenMV
- 控制算法：PID、逆运动学、步进电机控制
- 开发工具：Keil MDK、STM32CubeIDE、CCS、Mission Planner、嘉立创EDA、SolidWorks、VsCode、Codex等
